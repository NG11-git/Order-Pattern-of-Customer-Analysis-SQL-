# Order-Pattern-of-Customer-Analysis-SQL-


# Project Overview

This project analyzes customer order patterns using SQL queries on a sample dataset (introtallent.swiggy). The goal is to identify duplicate records, order trends, outlet‑wise performance, and customer‑wise sales insights.

# Dataset
* Table Name: introtallent.swiggy

* Columns:

1 id (INT)

2 cust_id (VARCHAR)

3 order_id (INT)

4 partner_code (INT)

5 outlet (VARCHAR)

6 bill_amount (FLOAT)

7 order_date (DATE)

8 comments (VARCHAR)
